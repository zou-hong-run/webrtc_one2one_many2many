# webrtc 一对一视频通话，多对多视频通话

下载项目

安装依赖

```
npm i
```
一对一通讯
```
/public/index.html 需要导入index.js
/index.js 下面需要导入 spd.js

```
多对多通讯
```
/public/index.html 需要导入index2.js
index.js 下面需要导入 spd2.js
```

运行项目

```
nodemon index.js
```
